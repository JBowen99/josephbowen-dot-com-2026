# josephbowen.com

Source code for my personal website — [josephbowen.com](https://josephbowen.com).

I'm Joseph Bowen, a product engineer with a background in full-stack software and biomedical engineering. This repo is a static Astro site for my portfolio: experience, personal projects, skills, and education.

## Local development

```sh
pnpm install
pnpm dev
```

## Deploy

The site builds to static files and is deployed via Docker (see `Dockerfile`).

```sh
pnpm build
```
